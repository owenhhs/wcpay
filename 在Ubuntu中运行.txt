现在在Ubuntu终端中运行以下命令：

----------------------------------------
步骤1: 安装git（如果还没安装）
----------------------------------------

sudo apt-get update
sudo apt-get install -y git

----------------------------------------
步骤2: 克隆项目
----------------------------------------

cd ~
git clone https://github.com/owenhhs/wcpay.git
cd wcpay

----------------------------------------
步骤3: 运行脚本
----------------------------------------

chmod +x scripts/*.sh
bash scripts/assist-test.sh

----------------------------------------
一键命令（复制全部）:
----------------------------------------

sudo apt-get update && \
sudo apt-get install -y git && \
cd ~ && \
git clone https://github.com/owenhhs/wcpay.git && \
cd wcpay && \
chmod +x scripts/*.sh && \
bash scripts/assist-test.sh
